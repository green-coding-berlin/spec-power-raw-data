System Under Test Notes
  • Each JVM instance was affinitized to a single core.
  • "input.load_level.number_warehouses" set to 8 in SPECpower_ssj.props to override "availableProcessors()" which in HotSpot returns the number of processors to which it is affinitized rather than total processors in the SUT;

  • "Adjacent Cache Line Prefetch" was disabled in BIOS.
  • "Hardware Prefetcher" was disabled in BIOS.
