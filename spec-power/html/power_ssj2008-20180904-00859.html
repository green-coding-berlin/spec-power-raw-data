System Under Test Notes
  • Manual disable unused Ethernet ports in the OS.
  • Using the local security settings console, "lock pages in memory" was enabled for the user running the benchmark.
  • SPECpower_ssj.props input.load_level.number_warehouses set to 128 due to a known inconsistency in processor reporting with this Java version.
  • The test was started via Remote Desktop.
  • Using Windows PowerShell, set the following powercfg properties:
    • powercfg -setacvalueindex scheme_current SUB_PROCESSOR CPMINCORES 100
    • powercfg -setacvalueindex scheme_current SUB_PROCESSOR CPPERF 0
