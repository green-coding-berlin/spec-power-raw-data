System Under Test Notes
  • Each JVM instance was affinitized to a pair of cores in a socket.
  • Huge pages were enabled in the OS using the following commands
    • echo "7100" > /proc/sys/vm/nr_hugepages
    • mount -t hugetlbfs nodev /mnt/hugepages
  • Disabled 'Hardware Prefetcher', 'Adjacent Cache Line Prefetch' and 'Snoop Filter" options in BIOS
  • Enabled C1E, EIST and variable Fan Speed Control options in BIOS
  • Power Managment Settings
    • Adjusted the CPU utilization threshold for P-state transitions to maximum frequency with: echo "90">/sys/devices/system/cpu/cpu*/cpufreq/ondemand/up_threshold (default 20)
    • Adjusted the CPU utilization sampling rate for P-state transitions with: echo "20000000">/sys/devices/system/cpu/cpu*/cpufreq/ondemand/sampling_rate (default 330000)
  • Benchmark was run via LAN
